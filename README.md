# GrammarChecker

A simple, rule-based application that checks grammar mistakes in input sentences.

To run [GrammarChecker], launch SWI-Prolog and load `checker.pl`, then run `q.` before each query.

```
$ swipl

?- [checker].
true.

?- q.
Type a sentence:
```
